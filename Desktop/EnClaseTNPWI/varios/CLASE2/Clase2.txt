CSS es el mecanismo que permite separar los contenidos definidos mediante XHTML, seria mas de:
 Base (Contenidos+presentacion) -> XHTML(contenidos) CSS(presentacion). 
Ventaja de separar ambos: es q los doc. creados son mas flexibles, ya que se adaptan mejor a dif. plataformas, pantallas de ordenador, pantallas de dispositivos móviles, impresoras y dispositivos utilizados por personas discapacitadas.
De esta forma, utilizando exclusivamente XHTML se crean páginas web "feas" pero correctas. AplicandoCSS, se pueden crear páginas"bonitas"a partir de las páginasXHTMLcorrectas.
